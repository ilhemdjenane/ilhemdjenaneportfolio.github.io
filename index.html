Amazon Store Sales Analysis and Dashboard

This project analyzes the Amazon Store's sales performance from 2005 to 2008, leveraging key metrics such as revenue, profit, transactions, and customer trends. A visually compelling dashboard was created to summarize insights, including quarterly and yearly trends, regional profitability, and weekday vs. weekend performance. The analysis provided actionable recommendations to optimize operations, increase profitability, and sustain growth. Tools used include Excel for data analysis and visualization. This project highlights my ability to transform raw data into meaningful insights and visually engaging reports, demonstrating my expertise in data storytelling and strategic decision-making.
